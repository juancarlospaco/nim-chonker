![](https://raw.githubusercontent.com/juancarlospaco/nim-chonker/master/temp.png "Does NOT run 'sudo mv', just prints the command for you")


# Compile

```
$ nim c chonker.nim
```

- Force writing to `/etc/pacman.d/chonker.conf` with `-d:overwrite`.
